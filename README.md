# notepad
<p>This is Notepad App for Android devices</p>
<p>Features:</p>
<p>- Text format panel</p>
<p>- Drawing panel</p>
<p>- Search notes</p>
<p>- Speech to text function</p>
<p>- Saving notes in SQLite database</p>
